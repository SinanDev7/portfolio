IT Support Engineer

Job Description at DHL

Company: DHL Global Forwarding (Sri Lanka)

Department: IT Infrastructure & Service Management

Employment Type: Outsourced (via SevenX Lanka Pvt Ltd

Duration: June 2024 - May 2025

Job Description

  • IT Support & ServiceNow: Delivered onsite and remote support for 120+ users across 3 remote locations. Handled 250+ incidents with 175+ resolutions via the GSN (Global ServiceNow) platform. Maintaining a 95% user satisfaction rate, including hardware, software, and network related issues.
  • IT Asset & Device Management (CMDB, ITHAM): Maintained 500+ assets in CMDB including laptops, desktops, monitors, printers, scanners, switches, routers, and mobile devices. Supported NGWS Managed Desktop and lifecycle updates.
  • OS Deployment (MECM): Deployed Windows OS to 100+ user machines using MECM task sequences via USB boot media, configuring OPE profiles and utilizing Adaptiva Peer Cache – boosting deployment speed by over 60% and reducing network bandwidth usage.
  • Printer & Scanner Management and Support: Supported and configured Ricoh, HP, Canon, SatoM, Zebra, Kodak, Alaris and other printers and scanners including toner replacement, cloud portal management, user access control, DNS record assignments, and cost tracking by department.
  • IT Infrastructure & Software Support: Provided IT Support for core business applications including Outlook, MS Teams, CHRIS, SalesForce, Sage50, Asycuda, Share Point and other M365 apps. Handled IP assignments (static/dynamic), configured PABX/IP phones, and set up mobile devices using Intune.
  • Telecom Asset & Mobile Connection Management: Managed mobile usage records and user inventory tracking in Excel. Maintained mobile number master files, generated monthly reports, and administered mobile connections via Dialog BizCare application including SIM activation/deactivations and data allocations.
  • Vendor Coordination & IT Procurement Support: Coordinated with vendors for warranty claims and technical support, assisted the IT Manager with new hardware procurement, and handled end-to-end vendor communication and follow-ups.
  • Meeting & Event IT Support: Participated in weely PD sessions, monthly Townhall and Open House meetings. Provided complete IT support including meeting room set up, projector, mic, speaker arrangements, remote UAT support, and technical assistance during special events and awareness meetings.
  • Project Support & Deployment: Provided IT and network support during the setup of DHL Centralized Warehouse Facility (Ekala Warehouse). Assisting major departments (IWS – IBM, HP, CISCO, Dell and LSH). Configured devices and supported the initial infrastructure setup for the Starlink implementation project by SpaceX, ensuring stable connectivity and system integration.
  • Membership & Collaboration: Contributed to internal DHL IT collaboration via GSN groups such as the ITIL User Group, Asset Management Group, and IT Support Group (DGF-LK-ITSupport), supporting process improvement and best practice sharing.
  • Tools & Software/ Applications Used: Microsoft Outlook, MS Teams, ServiceNow (GSN/CMDB), Active Directory Users & Computers, Microsoft Excel, Power Point, Word, Bomgar, MSRA (Microsoft Remote Assistance), Microsoft Intune, Zscaler, DHL Internal Applications (DHL Admin Portal, DHL Mobile Hub, DHL Request IT, DHL ChatGPT, DHL OPE Portal, DHL OSD Wizard, and more), Dialog BizCare, PuTTY, PIAM, Rufus, and other IT support tools and platforms.
