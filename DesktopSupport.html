Desktop Support Technician

Job Description at ACS Computers

Company: ACS Computer Solutions - Sri Lanka

Duration: Jan 2012 - Nov 2013

Job Description

  • Assembled and maintained desktop computers & laptops to meet client specifications
  • Installed and configured various operating systems, including Windows 7,8, and Server editions.
  • Installed and configured hardware components, software applications, and system drivers.
  • Set up and managed ADSL and wireless routers for SOHO (Small Office/Home Office) networks.
  • Performed server set up configuration, and basic maintenance (Windows Server Environments).
  • Diagnosed and resolved hardware & software issues to minimize downtime and support efficiency.
  • Managed installation and configuration of perepherals such as printers, scanners, and external drives.
  • Ensured all system setups met performance and usability standards based on user requirements.
